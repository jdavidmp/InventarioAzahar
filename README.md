# Inventario Azahar

Sistema de gestión de inventario desarrollado con:

- Node.js
- Express
- MySQL
- React (Vite)
- Generación de reportes PDF con PDFKit

## Funcionalidades

- Gestión de productos
- Registro de ventas
- Registro de compras
- Control de stock automático
- Anulación de ventas
- Reportes PDF filtrados por fecha
- Dashboard con estadísticas

## Reportes

- Reporte de ventas por rango de fechas
- Reporte de compras por rango de fechas
- Total general calculado automáticamente

##  Tecnologías utilizadas

Backend:
- Express
- MySQL
- PDFKit
- JWT (si usas autenticación)

Frontend:
- React
- Axios
- CSS
